# مصحف الركوع

تطبيق ويب (PWA) لقراءة القرآن الكريم وفق تقسيم الركوع.

يقسّم المصحف الشريف كاملًا إلى **٥٥٦ صفحة**، كل صفحة تمثّل ركوعًا واحدًا كاملًا حسب علامات الركوع (ع) الموجودة في المصحف، من الفاتحة إلى الناس. النص Uthmani من مصادر Tanzil / QPC (رواية حفص عن عاصم).

يعمل بالكامل بدون إنترنت بعد أول فتح (PWA)، وفيه شاشة رئيسية تعرض تقدّم القراءة، وفهرس للسور، وفهرس للأجزاء، وبحث، ومفضّلة لحفظ أي ركوع باسم مخصّص، وضبط لحجم الخط، ووضع ليلي.

## الطريقة الأسرع: تثبيته كتطبيق مباشرة (بدون بناء APK)

1. ارفع محتويات هذا المجلد بالكامل على أي استضافة ثابتة (استضافتك، أو GitHub Pages، أو Netlify — مجاني وسريع).
2. افتح الرابط من هاتف Android بمتصفح Chrome.
3. Chrome هيعرض تلقائيًا خيار **"تثبيت التطبيق" / Add to Home screen**.
4. بعد التثبيت، هيظهر التطبيق بأيقونته الخاصة زي أي تطبيق عادي، بشاشة كاملة، ويشتغل أوفلاين.

هذه أسرع وأبسط طريقة، ومفيش أي بناء أو تجميع مطلوب.

## لو محتاج ملف APK حقيقي فعلاً

### 1) PWABuilder (الأسهل — من غير أي كود)
1. ارفع المجلد على أي استضافة (نفس الخطوة اللي فوق) عشان يبقى ليه رابط عام.
2. روح على **https://www.pwabuilder.com**
3. حط رابط الموقع، دوسّ Start.
4. اختار Android من قائمة الباكدجات، وحمّل ملف الـ APK/AAB جاهز خلال دقايق.

### 2) Capacitor + Android Studio (لو عايز تحكم كامل في الكود الأصلي)
```bash
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init "مصحف الركوع" "com.alnas.mushafruku"
# انسخ محتويات هذا المجلد داخل مجلد www
npx cap add android
npx cap open android
```
وبعدين تبني الـ APK من جوه Android Studio (Build > Build APK) على جهازك اللي عليه Android SDK.

## هيكل الملفات
```
index.html      شاشة التطبيق الرئيسية (الرئيسية + القراءة + كل اللوحات)
style.css       التصميم (طراز صفحة مصحف)
app.js          منطق التنقل والفهرس والبحث والمفضلة والإعدادات
juz-info.js     بيانات العنوان والوصف
data.js         نص القرآن الكريم كاملًا مقسّم حسب الركوع (٦٢٣٦ آية / ٥٥٦ ركوعًا)
manifest.json   إعدادات PWA (اسم، أيقونة، ألوان)
sw.js           Service Worker للعمل أوفلاين
icons/          أيقونات التطبيق
```

## مصدر البيانات
- نص القرآن: حزمة `quran-json` (نص عثماني موثّق من The Noble Qur'an Encyclopedia / Tanzil).
- حدود الركوعات والأجزاء: حزمة `quran-meta` (بيانات Tanzil metadata الموثّقة لتقسيمات الركوع/الجزء/الحزب).

## رؤية المشروع

يهدف "مصحف الركوع" إلى توفير تجربة قراءة تعتمد على الركوع كوحدة مستقلة للورد اليومي، مع واجهة بسيطة، وسرعة عالية، وإمكانية العمل بالكامل دون اتصال بالإنترنت.

وسيتم تطوير التطبيق تدريجيًا حتى يشمل ميزات إضافية تساعد على القراءة والمراجعة والحفظ.
