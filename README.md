# جزء عمّ — بالركوعات

تطبيق ويب (PWA) لقراءة جزء عمّ مقسّمًا **٣٩ صفحة**، كل صفحة تمثّل ركوعًا واحدًا كاملًا حسب علامات الركوع (ع) الموجودة في المصحف الشريف. النص Uthmani من مصادر Tanzil / QPC (رواية حفص عن عاصم).

يعمل بدون إنترنت بعد أول فتح، وله فهرس للتنقل السريع بين السور، وضبط لحجم الخط، ووضع ليلي.

---

## الطريقة الأسرع: تثبيته كتطبيق مباشرة (بدون بناء APK)

1. ارفع مجلد `dist` بالكامل على أي استضافة ثابتة (استضافتك في المستشفى، أو GitHub Pages، أو Netlify — مجاني وسريع).
2. افتح الرابط من هاتف Android بمتصفح Chrome.
3. Chrome هيعرض تلقائيًا خيار **"تثبيت التطبيق" / Add to Home screen**.
4. بعد التثبيت، هيظهر التطبيق بأيقونته الخاصة زي أي تطبيق عادي، بشاشة كاملة، ويشتغل أوفلاين.

هذه أسرع وأبسط طريقة، ومفيش أي بناء أو تجميع مطلوب.

---

## لو محتاج ملف APK حقيقي فعلاً

⚠️ ملاحظة صريحة: تجميع (compile) ملف APK حقيقي محتاج Android SDK / Gradle، وده مش متاح في بيئة التنفيذ اللي بنيت فيها التطبيق (الشبكة هنا مقتصرة على مصادر كود مفتوح زي npm/pypi/GitHub فقط، مفيش وصول لسيرفرات Google/Android). فمقدرتش أطلعلك APK مُوقّع جاهز من هنا مباشرة. لكن في طريقتين سهلين تعملهم بنفسك:

### 1) PWABuilder (الأسهل — من غير أي كود)
1. ارفع مجلد `dist` على أي استضافة (نفس الخطوة اللي فوق) عشان يبقى ليه رابط عام.
2. روح على **https://www.pwabuilder.com**
3. حط رابط الموقع، دوسّ Start.
4. اختار Android من قائمة الباكدجات، وحمّل ملف الـ APK/AAB جاهز خلال دقايق.
5. لو عايزها موقّعة بمفتاحك الخاص عشان ترفعها على Google Play، الأداة بتديك خيار توليد Keystore.

### 2) Capacitor + Android Studio (لو عايز تحكم كامل في الكود الأصلي)
عندك خبرة في التطوير، فدي الطريقة اللي هتديك مشروع Android حقيقي تقدر تعدّل فيه:
```bash
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init "جزء عم" "com.alnas.juzamma"
# انسخ محتويات مجلد dist داخل مجلد www
npx cap add android
npx cap open android
```
وبعدين تبني الـ APK من جوه Android Studio (Build > Build APK) على جهازك اللي عليه Android SDK.

## هيكل الملفات
```
dist/
  index.html      شاشة التطبيق الرئيسية
  style.css       التصميم (طراز صفحة مصحف)
  app.js          منطق التنقل والفهرس والإعدادات
  data.js         نص جزء عمّ الكامل مقسّم حسب الركوع (٥٦٤ آية / ٣٩ ركوعًا)
  manifest.json   إعدادات PWA (اسم، أيقونة، ألوان)
  sw.js           Service Worker للعمل أوفلاين
  icons/          أيقونات التطبيق
```

## مصدر البيانات
- نص القرآن: حزمة `quran-json` (نص عثماني موثّق من The Noble Qur'an Encyclopedia / Tanzil).
- حدود الركوعات: حزمة `quran-meta` (بيانات Tanzil metadata الموثّقة لتقسيمات الركوع/الجزء/الحزب).
